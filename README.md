# lstemu

## Description
Tools and example notebooks to assign and use event classes for CTA-compliant IACT data sets (e.g. CTA/LST).

## Installation

You may consider setting up a virtual environment to install the package to e.g. as

```bash
python3 -m venv lst-irf-classes
source lst-irf-classes/bin/activate
```

Then clone and install the package with `pip`:

```bash
git clone https://github.com/IevgenVovk/lst-irf-classes.git
cd iclass
pip install .
```
